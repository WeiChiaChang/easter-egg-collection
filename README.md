## ğŸ‰ Installation

In a browser (CDN):
```html
<script src="https://rawgit.com/WeiChiaChang/Easter-egg/master/easter-eggs-collection.js"></script>
```

Or using npm:

```shell
npm i easter-egg-collection -D
```

Then in your JavaScript file:

```javascript
import collection from 'easter-egg-collection'
```
